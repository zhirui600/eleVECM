# 广东电力现货市场日前电价影响因素研究

本项目旨在通过 **向量误差修正模型 (VECM, Vector Error Correction Model)** 实证研究和分析广东电力现货市场中日前电价的核心影响因素。项目基于 Python 环境，主要使用 `statsmodels` 等计量经济学和时间序列分析库进行建模。

## 📊 变量说明

本研究综合考虑了供给侧、需求侧、气象及外部输入等多个维度的变量：

*   **`electrovalence`**：电力日前电价 (核心被解释变量)
*   **`x_coal`**：煤炭价格指标
*   **`x_wind`**：风力发电出力/风电预测
*   **`x_temp`**：气温指标
*   **`x_eleWest`**：西电东送电量
*   **`x_demand`**：全社会用电需求/负荷
*   **`x_lng_1`**：LNG（液化天然气）价格
*   **`x_df`**：其他相关调峰/市场控制变量

## 🔬 研究方法与分析流程

本项目在 `VECM.ipynb` 中实现了完整的时间序列与协整分析流程：

1.  **数据加载与预处理**：处理异常值，并对所有变量进行标准化 (StandardScaler) 处理，消除量纲影响，提高数值稳定性和系数的可解释性。
2.  **Johansen 协整检验 (Johansen Cointegration Test)**：运用迹检验 (Trace Test) 分析多个时间序列之间是否存在长期稳定的均衡（协整）关系，并计算出协整秩 (Cointegration Rank)。
3.  **最优滞后阶数选择**：基于 VAR 系统的各种信息准则 (AIC, BIC, HQIC, FPE)，确定模型最优的 VAR 滞后阶数，进而得到 VECM 差分项的滞后阶数。
4.  **VECM 模型构建与估计**：构建向量误差修正模型，分离并量化各影响因素对日前电价的短期波动影响与长期均衡约束。
5.  **系统稳定性检验**：利用伴随矩阵 (Companion Matrix) 特征值检验 VECM 系统是否满足平稳性要求。
6.  **脉冲响应函数分析 (IRF)**：追踪并可视化某一特定因素（如煤价上涨、气温突变或风电骤减）产生单位标准差冲击时，系统及电价在未来若干期内的动态响应轨迹。
7.  **预测误差方差分解 (FEVD)**：量化各变量的结构性冲击对日前电价预测方差的相对贡献度，找出驱动电价波动的主导因素。
